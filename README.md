# expert-notes

## Project Setup

1. Clone the repository
    ```sh
    git clone git@github.com:m-piovesan/nlw-expert-notes.git
    ```
2. Navigate to the project directory
    ```sh
    cd nlw-expert-notes
    ```
3. Install dependencies
    ```sh
    npm install
    ```

## Running it

### Compiles and hot-reloads for development
```
npm run dev
```
